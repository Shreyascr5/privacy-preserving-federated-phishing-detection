# Privacy-Preserving Federated Learning for Phishing URL Detection

## Overview

This project proposes a lightweight and privacy-preserving phishing URL detection framework using Federated Learning (FL) and Edge AI.

## Objectives

- Detect phishing URLs using machine learning
- Preserve user privacy through federated learning
- Reduce communication overhead using selective updates
- Support deployment on resource-constrained edge devices

## Technologies

- Python
- Flower
- Scikit-Learn
- Streamlit
- Pandas
- NumPy

## Project Phases

1. Dataset Collection & Preprocessing
2. Baseline Model Development
3. Federated Learning Implementation
4. Differential Privacy Integration
5. Selective Update Mechanism
6. Streamlit Dashboard
7. Experimental Evaluation
